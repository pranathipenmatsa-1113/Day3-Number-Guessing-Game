# Number Guessing Game using Python

## Project Description

This is a simple Python Number Guessing Game developed as part of my Python learning journey.

The program stores a secret number and asks the user to guess it. Based on the user's input, the program provides hints indicating whether the guessed number is too high, too low, or correct.

## Features

- User input handling
- Conditional statements (if, elif, else)
- Number comparison
- Simple game logic
- Beginner-friendly project

## Technologies Used

- Python

## Concepts Practiced

- Variables
- Data Types
- Input and Output
- Conditional Statements
- Logic Building

Guess a number: 11
Too High

Guess a number: 10
Correct Guess!
